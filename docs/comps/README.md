# 安装

## npm安装

开箱即用 无外网直接将文件夹拷贝到node_modelus中即可使用 , 待后期建立npm或者git仓库后可在外网使用 npm install vue-hangtian-ui 使用

```
npm i vue-hangtian-ui -S
```

## 使用

本框架是基于element-ui 2.15.7版本进行的二次封装组件，意味着我们可以直接使用element-ui 2.15.7的所有组件， 可以直接参照element-ui官网 ,

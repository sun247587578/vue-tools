
# 标题

## ht-h1

一般用来当标头使用

::: demo 
```html
<template>
  <div class="map-views pd10 ">
      <ht-h1>测试标题</ht-h1>
      
    </div>
</template>
```
:::

::: warning
因为标头的特殊形状 所以请注意宽度最好不要低于260px
:::

## ht-h3

> 倾斜背景默认样式
::: demo 
```html
<template>
  <div class="map-views pd10 ">
    <ht-h3>倾斜背景的标题样式</ht-h3>
  </div>
</template>
```
:::

> 倾斜背景聚焦样式
::: demo 
```html
<template>
  <div class="map-views pd10 ">
    <ht-h3 :active="true">倾斜背景的标题样式</ht-h3>
  </div>
</template>
```
:::
